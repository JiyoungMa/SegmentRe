# SegmentRe

<h2> 4학년 1학기 협동분산시스템 강의에서 진행한 팀 프로젝트 결과물인 'Segment'의 서버와 클라이언트를 다시 만들어보는 프로젝트 </h2>
<h2> A project to recreate the server and client of "Segment", the result of the team project conducted in the first semester of the fourth grade. </h2>

- Spring Boot, JPA, Thymeleaf, MySQL
- 현재는 쿠키를 사용 => **추후에 세션으로 변경 예정**
- Currently Using Cookies => **Will be changed to session in the future**
